THE 30-DAY ROADMAP

Four weeks. Four superpowers.

Tap any module to see what's inside. Every week ends with a project you can show your parents.

01 WEEK ONE Welcome to the Matrix

Break down the mystery of "coding" with everyday analogies — recipes, cricket umpire rules, autorickshaw routes. Tour of websites, apps, games and AI — all just code, nothing scary.

Mini projectWrite a program (in plain English!) to make Maggi — your first algorithm.

Set up VS Code on your laptop. Learn the interface, install pro extensions (Prettier, Live Server, Python), master 5 keyboard shortcuts that make you look like a hacker.

Mini projectCreate your first hello.txt and own your dev setup.

Learn 8 terminal commands that unlock your whole machine. Once you can navigate with the keyboard, computers stop being mysterious.

Mini projectBuild a folder structure using ONLY the terminal — no mouse.

Create your GitHub account. Learn git init, add, commit, push. Apply for the free GitHub Student Pack (₹15,000+ of free developer tools for school students).

Mini projectShip your first public repo with a custom README.

Build a real professional profile — the right headline, the right photo, the right first 20 connections. A head start most college students don't even have yet.

Mini projectGo live with your profile + your first #HelloLinkedIn post.
Week 1 unlock: Real dev setup + your first GitHub commit + a live LinkedIn profile.
02 WEEK TWO Build your first website

Headings, paragraphs, images, links, lists, forms. Learn how the browser reads your code & see instant results on every save.

Mini projectAn "About Me" page with your photo, bio, favorites & links.

Google Fonts, color palettes, Flexbox, the box model, and responsive design for mobile. Where HTML teaches structure, CSS unlocks creativity.

Mini projectRestyle your "About Me" with gradients, hero section & mobile layout.

Use GitHub Pages to host your site — 100% free, no credit card, real URL. This is the session that makes everyone call their mom.

Mini projectPortfolio live at yourname.github.io — forward it to the family group.
Week 2 unlock: Your portfolio website is LIVE on the internet. 🎉
03 WEEK THREE Enter the Python

Install Python. Run your first script three different ways — terminal, VS Code, and browser (Replit, Google Colab).

Mini projectA script that introduces YOU to the world.

Numbers, text, booleans, lists. Take user input and print smart responses. The grammar of programming.

Mini projectA CBSE Marksheet Calculator — total, %, auto-grading.

Conditionals, comparisons, logical operators — the first taste of "computational thinking."

Mini projectBhilai Auto Fare Calculator with surge-pricing rules.

For loops, while loops, and reusable functions. The real "aha, computers ARE fast" moment.

Mini projectBuild & ship a Guess-the-Number game + a 5-question Quiz App.
Week 3 unlock: 3 working Python projects in your GitHub toolbox.
04 WEEK FOUR You vs AI (spoiler: you win)

What ChatGPT actually does. Why it hallucinates. Why it's not Google. Kid-friendly intro aligned with UNESCO's AI framework for students.

Mini project"Hallucination Hunt" — trick AI into being confidently wrong & learn why.

When to use which tool. ChatGPT for code, Claude for writing, Gemini for research (free Pro for students!), NotebookLM for studying from your own notes.

Mini projectSolve the same physics problem on all 4 — build your personal AI cheat sheet.

Role + Task + Context + Constraints + Format. Few-shot prompting. Chain-of-thought. Use AI's Study Modes to learn, not just answer.

Mini projectA 10-prompt personal library for summarizing, explaining, debugging & more.

"AI as tutor, not ghostwriter." Academic integrity, bias, privacy. Walk away with a real code of conduct.

Mini project"Ethics Court" — 4 live scenarios debated & voted in class.

Learn what an API is. Call a real AI model with Python. Design your bot's personality (study buddy? cricket expert? motivation coach?). Give it a web interface.

Mini projectA deployed chatbot with YOUR personality — show it on Parents' Demo Day.
Final unlock: Your own AI + website + GitHub + LinkedIn. Demo Day with parents. 🎉
